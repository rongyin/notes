## MySQL命令

### 管理命令
#### 连接到数据库
* -h 指定连接数据库ip地址
* -u 指定登录数据库用户名
* -p 登录需要输入密码

```
mysql -h host_name -u user_name -p
```
#### 列出数据库实例
```
mysql>SHOW DATABASES;
```
#### 选择连接数据库实例
```
mysql>USE instance_name;
```
#### 列出实例的所有表
```
mysql>SHOW TABLES;
```
#### SHOW COLUMNS
```
mysql>SHOW COLUMNS FROM table_name;
```
#### SHOW INDEX
```
mysql>SHOW INDEX FROM table_name;
```
#### SHOW TABLE STATUS
```
mysql>SHOW TABLE STATUS [FROM instance_name][LIKE 'pattern']\G;
```
